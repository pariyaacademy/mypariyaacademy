System Administrator Guide

📋 Google Sheet Column Structure

Required Column Names

Your Google Sheet must contain these exact column headers:

  • Name — Student's full name
  • Admission No — Unique admission number (used for student login)
  • Class Type — e.g. "JSS 2", "Primary 3"
  • Class Category — e.g. "Junior Secondary", "Primary", "Nursery", "Senior Secondary"
  • Class Section — e.g. "A", "B" or blank
  • Gender — "Male" or "Female"
  • Term — "First Term", "Second Term", or "Third Term"
  • Session — e.g. "2025/2026"
  • Total Term Days, Days Present, Days Absent
  • Resumption Date — e.g. "12 Sept. 2026"
  • SubjectName_CA1, SubjectName_CA2, SubjectName_Exams — For Primary/Nursery
  • SubjectName_CA1, SubjectName_CA2, SubjectName_CA3, SubjectName_Exams — For JSS/SS

Subject Column Format

Subject scores must follow this pattern: Mathematics_CA1, Mathematics_CA2, Mathematics_Exams

The part before the underscore (Mathematics) becomes the subject name on the report card.

🔗 Publishing Google Sheet as CSV

How to Get CSV URL

  • Open your Google Sheet
  • Click File → Share → Publish to web
  • Under "Link", choose the correct sheet tab
  • Change format from "Web page" to Comma-separated values (.csv)
  • Click Publish
  • Copy the link that appears — this is your CSV URL

Where to Paste CSV Links in Code

Find the CLASS_LINKS object in the JavaScript section. Replace the placeholder URL for each class:

"JSS 2": "https://docs.google.com/spreadsheets/d/e/..."

Make sure the URL ends with &output=csv

⚙️ Editing Classes in Code

Add or Remove Classes

Edit the CLASS_LINKS object to add or remove classes. Each key is the class name, each value is the CSV URL.

Also update CLASS_META with the icon and category for each class (nursery, primary, jss, ss).

🔐 Editing Credentials

Admin and Student Passwords

Find the CREDENTIALS object in the JavaScript section:

admin: { username: "admin", password: "pas2017" }

student: { password: "student123" }

Change these values to your desired username/password. All students share the same password.

🎨 Editing Colors in Code

Dashboard Colors

Find the :root CSS block at the top. Variables starting with --dash- control the dashboard colors. Do NOT change variables starting with --primary, --soft, --accent, --muted, --border — these control report card colors.

Class-Specific Report Card Colors

Find the section labeled "REPORT CARD — CLASS-SPECIFIC COLOURS". Each class category (nursery, jss, ss) can have its own color theme.

📝 Editing Report Card Text

School Name, Motto, Address

Find the SCHOOL_CONFIG object and edit: name, motto, address, session, resumption.

Authority Names and Signatures

Edit headmasterName, headmasterSig, principalName, principalSig, primaryFormMasterName, secondaryFormMasterName in SCHOOL_CONFIG.

Signature images should be hosted on PostImage or similar and the direct link pasted here.

📚 How to Add Subjects

Subjects are Auto-Detected from Sheet

You do NOT need to edit code to add subjects. Simply add new columns to your Google Sheet following the pattern: SubjectName_CA1, SubjectName_CA2, SubjectName_Exams.

The system automatically detects all subject columns and adds them to the report card.

🌐 Deploying on Netlify

Step-by-Step Deployment

  • Go to netlify.com and sign up / log in
  • Click "Add new site" → "Deploy manually"
  • Drag and drop your index.html file (rename it to index.html if needed) into the upload box
  • Netlify will give you a public URL immediately
  • To update: go to your site dashboard → Deploys → drag new file
  • For a custom domain: go to Domain settings and follow the instructions

🌐 School Websites Manager

Adding Websites via Google Sheet

Create a Google Sheet with columns: Function, URL, Username, Password.

Publish it as CSV and paste the link into the WEBSITES_CSV_URL variable in the config section.

Credentials are hidden by default — admin clicks "Show" to reveal them.

👨‍🏫 Teachers Sheet Structure

Required Column Headers (Teachers CSV)

Your Teachers Google Sheet must use these exact column headers for the system to load teacher records correctly:

  • TeacherName — Full name of the teacher (e.g. "Mal. Ibrahim Abubakar")
  • ClassCategory — Section the teacher belongs to. Accepted values: Nursery, Primary, Junior Secondary, Senior Secondary. Multiple sections can be separated by commas (e.g. Primary, Junior Secondary)
  • Subjects — Subjects the teacher teaches (e.g. "Mathematics, English")
  • ClassType — The specific class assigned (e.g. "JSS 2A", "Primary 3")
  • Qualification — Academic qualification (e.g. "B.Ed Mathematics")
  • Phone — Contact phone number
  • Salary — Monthly salary figure (shown in teacher profile)
  • Month1, Month2, Month3 — Payment columns per term (First Term: Jan/Feb/Mar, etc.)

Alternative / Legacy Column Names Supported

The system also recognises these variations if you already have a sheet with different headers:

  • Teacher Name (with space) instead of TeacherName
  • Section instead of ClassCategory
  • Class Type (with space) instead of ClassType
  • Class Section instead of ClassSection

Section Abbreviations Accepted

In the ClassCategory or Section column you may use single-letter abbreviations:

  • n → Nursery
  • p → Primary
  • j → Junior Secondary
  • s → Senior Secondary

How to Publish the Teachers Sheet as CSV

  • Open your Teachers Google Sheet
  • Click File → Share → Publish to web
  • Select the correct sheet tab and choose CSV format
  • Click Publish and copy the link
  • Paste the link into TEACHERS_CSV_URL in the JavaScript config section

Why Teachers Section Shows Empty

If you open the Teachers section and see no teachers, check these common causes:

  • The TEACHERS_CSV_URL is not set or is pointing to an unpublished sheet
  • The sheet uses different column headers than listed above
  • The ClassCategory or Section column values don't match the section you clicked
  • The sheet has not been published to web with CSV output

🏆 Awards & Certificates

Best Student Awards (Automatic)

Go to Analytics → Load All Analytics first. The system computes the annual average (all 3 terms) for every student in every class and identifies the best student in each section (Nursery, Primary, Junior Secondary, Senior Secondary). Then go to Awards → Best Student Award and click Generate — certificates are produced automatically for all 4 sections and can be downloaded as landscape PDF.

Custom Student Award

Fill in the student name, class, section, and award title/purpose manually, then click Generate. Use this for any special recognition (e.g. "Most Improved Student", "Best in Mathematics").

Teacher Award

Fill in the teacher's name, role, section, and award purpose. Click Generate to produce a landscape blue-themed staff certificate with the principal's and head teacher's signatures.

Certificate Signatures

Certificates use the authority signatures configured in SCHOOL_CONFIG: headmasterSig and principalSig for Nursery/Primary and Secondary sections respectively, and primaryFormMasterSig / secondaryFormMasterSig for form masters. Upload signature images to PostImage or similar and paste the direct links in SCHOOL_CONFIG.
